# 深海このか JPYC投げ銭ページ 🍀

深海このかを応援するためのJPYC投げ銭システムです。

## 🎨 機能

- Web3ウォレット連携（MetaMask対応）
- Polygon Networkで動作
- JPYCトークンで投げ銭可能
- 応援メッセージ付き

## 🚀 使い方

1. [投げ銭ページ](https://nounsgirl.github.io/jpyc-tip-jar/)にアクセス
2. MetaMaskを接続
3. 金額を選択
4. メッセージを入力（任意）
5. チップを送る

## 💡 必要なもの

- MetaMask（ブラウザ拡張機能）
- Polygon Network上のJPYC残高

## 📝 技術スタック

- HTML/CSS/JavaScript
- ethers.js v5
- JPYC Token on Polygon
